# Notes - Draggable Cards Note Taking Application

This project is a simple and elegant note-taking web app that allows users to create draggable and resizable note cards in the browser. It features customizable light and dark themes, adjustable background opacity, and rich text editing options.

## Features

- Draggable and resizable note cards
- Editable card titles and content with rich text formatting
- Toggle between light (day) and dark (night) themes
- Adjustable background opacity slider
- Text formatting: bold, italic, underline, strikethrough
- Change font color and highlight color
- Select font size from dropdown
- Changes automatically saved in localStorage to persist between sessions
- Settings menu for theme, opacity, and adding new cards
- Contextual toolbar for formatting text upon selection

## Usage

1. Click the gear icon at the bottom-right to open the settings menu.
2. Toggle between day and night mode using the "Theme" option.
3. Adjust background opacity with the slider.
4. Add new note cards with "Add New Card".
5. Click and edit card titles or contents directly.
6. Select text inside a card to reveal the formatting toolbar.
7. Drag cards anywhere on the screen or resize using the corner handle.
8. Delete cards with the Ã— button on the card header.
